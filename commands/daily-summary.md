---
description: 生成今日工作总结（9:00 - 18:00 工作时间内的提交记录）
---

1. **读取今日提交记录**：
   - 读取文件 `~/logs/$(date +%Y-%m-%d)/commits.txt`
   - 如果文件不存在，提示用户今天没有提交记录

2. **筛选工作时间内的提交**：
   - 只包含 09:00 到 18:00 之间的提交
   - 解析每条提交记录的时间戳

3. **按项目分组整理**：
   - 将提交按项目路径分组
   - 合并同一项目的多次提交

4. **生成工作总结**：
   - 使用简洁的中文描述
   - 格式示例：
     ```
     ## 今日工作总结 (YYYY-MM-DD)

     ### 项目名称
     - 完成了 xxx 功能
     - 修复了 xxx 问题

     ### 另一个项目
     - 添加了 xxx 支持
     ```

5. **统计信息**：
   - 显示总提交次数
   - 涉及的项目数量
   - 工作时间段分布

注意事项：
- 如果提交时间格式不规范（如包含 $(date) 等），尝试从上下文推断
- 合并重复的提交记录（相同 commit hash）
- 重点突出功能性改动（feat, fix），弱化文档和配置更新
